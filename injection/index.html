Medium SQL Injection Challenge

Your goal: log in as admin and get the flag.

The backend is a vulnerable login system using SQLite. You don't know the admin password.

Known information (leaked by a careless developer): 

Table: users
Columns:
  id        INTEGER PRIMARY KEY
  username  TEXT
  password  TEXT
  is_admin  INTEGER (0 or 1)
  flag      TEXT (only for admin)

Try typical SQL injection payloads in the username / password fields to bypass the login and become admin.
